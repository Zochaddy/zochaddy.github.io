https://zochaddy.github.io

## GitHub Repository

https://github.com/Zochaddy/zochaddy.github.io

## How to Run

1. Clone the repository: `git clone https://github.com/Zochaddy/zochaddy.github.io`
2. Open `index.html` in a web browser, no installation or build steps required
3. Or visit the live site at https://zochaddy.github.io

## Outside Libraries and Frameworks

- **Bootstrap 5.3.3** (https://getbootstrap.com) — used for navbar, cards, accordion, form styling, and grid layout
- **Google Fonts - Knewave** (https://fonts.google.com/specimen/Knewave) — used for page title and navbar font

## Outside Resources

- **Bootstrap Accordion Docs** (https://getbootstrap.com/docs/5.3/components/accordion/) — referenced for building the accordion component in the Relevant Coursework section
- **W3Schools - background-attachment** (https://www.w3schools.com/cssref/pr_background-attachment.php) — referenced for `background-attachment: fixed` parallax effect
- **MDN - background-attachment** (https://developer.mozilla.org/en-US/docs/Web/CSS/background-attachment) — another reference for `background-attachment: fixed`
